# 🚚 SmartLogistics AI

An AI-powered Smart Delivery & Logistics System built using Streamlit, Python, and Machine Learning.

---

## 🔥 Features


- 📦 Order Booking System
- 🔍 Live Order Tracking
#- ⏱️ ETA (Estimated Delivery Time)
- 📊 Admin Dashboard & Analytics
- 🚴 Delivery Partner Panel
- 📱 WhatsApp Notification (Twilio)
#- 🗺️ Map Integration   # Not avilable
- 🌦️ Weather Awareness (optional)
- 🤖 AI-based Delay Prediction (coming soon)

- Python 🐍
